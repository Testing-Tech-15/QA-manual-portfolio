# CL-001: Dashboard QA Checklist

- [ ] Verify only current user data appears in Dashboard  
- [ ] Log in as different users to check isolation  
- [ ] Refresh page and ensure old content disappears  
- [ ] Confirm no sensitive data is visible from other accounts  
- [ ] Document all anomalies in Bug Reports



# CL-002: Extension Install Checklist

- [ ] Verify "Install Now" button redirects correctly  
- [ ] Confirm extension loads as expected in Chrome Web Store  
- [ ] Test on multiple browsers (Chrome, Edge, Opera)  
- [ ] Check UI messages in case of redirect error  
- [ ] Document any discrepancies in Bug Reports
